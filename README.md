# Electric Vehicle Analysis Dashboard ⚡🚗

## 📌 Project Overview

This project focuses on analyzing **Electric Vehicle (EV) Population Data** using **Power BI**. The main objective of this project is to understand the growth of electric vehicles, analyze market trends, and gain insights into different EV manufacturers, vehicle types, and geographical distribution.

The dashboard helps in identifying patterns and provides useful business insights through data visualization.

---

## 🎯 Project Objective

The objective of this project is to analyze electric vehicle adoption trends and answer important business questions such as:

* Which EV brands are most popular?
* Which cities/counties have the highest EV adoption?
* What is the distribution of EV types (BEV/PHEV)?
* How has EV adoption changed over different model years?
* What is the electric range offered by different vehicles?

---

## 🛠 Tools & Technologies Used

* **Power BI** → Dashboard creation and visualization
* **CSV Dataset** → Data source for analysis
* **Excel / Power Query** → Data cleaning and transformation
* **DAX (Data Analysis Expressions)** → Measures and calculations

---

## 📂 Dataset Information

The dataset contains **150,000+ electric vehicle records** with the following important fields:

* Vehicle Make
* Vehicle Model
* Model Year
* Electric Vehicle Type
* Electric Range
* Base MSRP
* State / City / County
* CAFV Eligibility
* Electric Utility Provider

---

## 📊 Dashboard Features

The Power BI dashboard includes the following analysis:

### 1. EV Distribution Analysis

* Total number of electric vehicles
* EV adoption across different locations

### 2. Manufacturer Analysis

* Top EV manufacturers by vehicle count
* Brand comparison analysis

### 3. Vehicle Type Analysis

* Battery Electric Vehicle (BEV) distribution
* Plug-in Hybrid Electric Vehicle (PHEV) distribution

### 4. Year-wise Trend Analysis

* Growth of electric vehicle adoption by model year
* Trend comparison over time

### 5. Range Analysis

* Comparison of electric range across different EV models
* Average range analysis

### 6. Geographical Analysis

* State-wise and city-wise EV distribution
* Regional adoption trends

---

## 📈 Key Insights

* Electric vehicle adoption has grown rapidly in recent years.
* Certain manufacturers dominate the EV market.
* BEV vehicles are more common than PHEV in the dataset.
* EV adoption is concentrated in specific regions.
* Modern EV models offer significantly improved electric range.
