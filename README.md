Electric Car Vehicle Analysis Dashboard
Project Overview
This project analyzes Electric Vehicle (EV) data using an interactive dashboard created in Power BI.
The main objective of this project is to understand electric vehicle trends, vehicle performance, and market insights through data analysis and visualization.
Objective
The purpose of this project is to analyze electric vehicle data and identify important insights such as:
Growth of electric vehicles in the market
Vehicle type distribution
Battery capacity comparison
Range analysis of electric vehicles
Manufacturer-wise vehicle analysis
Trend analysis and performance comparison
